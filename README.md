# 🔐 Login & Signup System (React)

A responsive authentication system built using React.js that allows users to register, log in, and manage authentication using token-based flow.

## 🚀 Features
* 🔑 User Registration (Signup)
* 🔐 User Login Authentication
* 📡 API Integration using Fetch API
* ✅ Form Validation (Email & Password)
* 💾 Token Storage using LocalStorage
* ⚠️ Error Handling (Network & API)
* 📱 Responsive UI Design

## 🛠️ Tech Stack
* **Frontend:** React.js
* **Styling:** CSS
* **State Management:** useState (React Hooks)
* **API Handling:** Fetch API

## 💡 How It Works
1. User enters email & password
2. Data is sent to backend API
3. On success → token is stored in localStorage
4. User can access protected features
